DISPLAY & VISIBILITY


-> every element is a box.
-> DISPLAY effects layout of neighbouring elements.

INLINE: 
        - sits next to other elements.
        - takes up just enough width & height 
        - rules cannot be set on height & width


BLOCK:
       - forces line break
       - takes up all horizontal width & just enough height 
       - rules can be set on height & width


INLINE-BLOCK:
       - same as inline 
       - height & width can also be set


NONE:
      - removed from page.


COMPLEMENTARY PROPERTIES

FLOAT:
      - reposition elements to left/rigth.
      - elements are aware of each other and will not overlap.
      - Values: left, right


CLEAR:
      - used to keep floating elements away.
      - Values: left, right, both


OVERFLOW

-> visible, hidden, scroll, auto


New DISPLAY Values

-> table, grid, flexbox

TABLE: - table-like layout without using table structure
       - can use "diplay:table" along with "display:table-cell" for elements.


VISIBILITY

-> visible, hidden
-> collapse (only for table elements)

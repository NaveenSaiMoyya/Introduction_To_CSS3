MEASUREMENTS

Absolute:  
         - set to a specific size.
         - px(pixel), mm(millimeter), cm(centimeter), pt(point) etc..

Fluid:
         - sets size relative to surrounding elements.
         - %(perecntage), vw(viewport width), vh(viewport height)
         - em (for font)  1 em => current size
                          .75em => 75% of current size
        -rem (for font)   1 rem => current size of root element

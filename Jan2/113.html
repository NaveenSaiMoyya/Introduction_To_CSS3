COLORS


PRIMARY COLORS : Red, Green, Blue

-> color names are not consistent across the Browsers
-> Hexadecimal - #xxxxxx or #xxx
               - x can be a value from (0,1,2,...,9,A,B,C,...F)
               - it can be of 3 digits or 6 digits
        ex:-
             #FF0000 - Red
             #00FF00 - Green
             #0000FF - Blue
             #FFFF00 - Yellow

-> RGB - (X,X,X)
       - X value can be in between (0,1) inclusive.

    ex:-
         (1,0,0) - Red
         (0,1,0) - Green
         (0,0,1) - Blue
         (1,1,0) - Yellow
-> RGBA - (X,X,X,X)
        - 4th X is alpha tranparency.

-> Keep meaningful colors.
-> color contrast should allow viewers to differentiate & able to read the content out of background color.
-> use web safe colors & use an accepted convention.
-> Test the site using Contrast Checker.
-> Don't get attached to some colors only.

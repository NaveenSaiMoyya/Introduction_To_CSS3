ACCESSIBILITY


Perceivable:

-> provide text alternatives for images.
-> provide captions & transcripts for video & audio.
-> use correct semantic markup.
-> good color contrast


Operable:

-> All the functionalities that are in a page should be available from keyboard.
-> users must have control over timing and limits (audio/video).
-> avoid using seizures (flash content).
-> provide ways to help users navigate.


Understandable:

-> economical & plain use of Language.
-> using good universal design.
-> Navigation,info structure must be discernable (recognize or find out) and consistent. 
-> making pages operated in predictable ways.
-> help users avoid & correct mistakes.


Robust:

-> is the site functional across various platforms (smartphones, screen readers, laptop, pensticks).
-> Responsive Design (works with different screen sizes).
-> syntax errors don't effect visual presentation but hampers assistive technology & accessibility tools.
-> adhering to W3C standards.

Styling The TEXT


Font  - - - font family
      - - - font style
      - - - font variant
      - - - font size


FONT FAMILY

-> courier, "courier new", "comic sans ms", cursive, verdana, Helvetica
-> All browsers doesn't support all font-families. we can provide alternatives.


-> Basically 2 categories of font-families : Serif, Sans-Serif (mostly recommended)


-> to use custom fonts, use @font-face



FONT STYLE

-> normal, italic, oblique


FONT VARIANT

-> normal, small-caps


FONT SIZE

->  xx-small, smaller, medium, small, x-small, larger, x-large, xx-large
-> mostly pixels & percentages are used instead of names.


COLOR

-> color of foreground (Text itself)


BACKGROUND-COLOR

-> color of background


TEXT-ALIGN

-> left, right, center, justify


LINE-HEIGHT

-> space b/w the lines
-> percentages
